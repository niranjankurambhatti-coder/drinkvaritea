# Campaign Library — Drink Varitea PMF Probes

First 7 PMF campaigns. Each campaign is a structured probe, not just an ad.

> **Priority order:** P0 campaigns launch first, sequentially, with results informing P1.

---

## Campaign 1 — Choice Overload

**Priority:** P0

| Field | Value |
|---|---|
| PMF question | Do people pay specifically to avoid tea choice overwhelm? |
| Target persona | Variety Seeker, Ritual Seeker (beginner stage) |
| Pain point | "Tea stores have 100 options. I don't know where to start." |
| Desire | Confident first purchase without research burden |
| Ad hook | "Tea stores have too many choices. We made it easy." |
| Landing page promise | The First Sip Box: 3 curated teas for people who don't know where to start. No guessing. |
| Search keywords | `beginner tea box`, `loose leaf tea starter kit`, `tea sampler for beginners`, `easy tea gift`, `best first tea` |
| Meta audience | Interests: tea + wellness; age 25–44; US; exclude existing followers |
| Creative needed | Founder Reel (founder talking to camera, problem/solution), product mockup static |
| CTA | Get the First Sip Box |
| Primary KPI | Checkout CTA click rate |
| Kill rule | After spending $[founder approval] with < 5 checkout CTA clicks total |
| Scale rule | CTR > 1.5%, CPC < $[founder approval], ≥ 1 purchase |
| utm_campaign | `pmf_choice_overload` |
