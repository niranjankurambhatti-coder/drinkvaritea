# Asset Requirements

All assets Brand and Social agents must produce before campaigns can run.

> **Rule:** Mark `Required Before Launch: YES` items as hard gates. No ads without them.

---

## Brand foundation assets

| Asset | Purpose | Spec | Copy Required | Owner | Required Before Launch |
|---|---|---|---|---|
| Logo (SVG + PNG) | All ads, landing pages, email headers | SVG master + PNG 512×512, 1024×1024, transparent bg | None | Brand Agent | YES |
| Color system | Consistent across all ads and landing pages | Primary, secondary, accent, background hex codes | None | Brand Agent | YES |
| Typography | Font pairing for headlines and body | Google Font or licensed font, specified weights | None | Brand Agent | YES |
| Brand voice guide | Copy tone for all ads and landing pages | 1-page doc: do/don't list, 3 example sentences | N/A | Brand Agent | YES |

## Asset production checklist before first ad spend

- [ ] Logo (SVG + PNG transparent)
- [ ] Color hex codes finalized
- [ ] Font specified
- [ ] First Sip Box hero mockup (1080×1080)
- [ ] Founder Reel 1 filmed and subtitled (15–30s)
- [ ] Square static ad with product + headline
- [ ] Google Search ad copy written (3 headlines + 2 descriptions)
- [ ] OG image for default landing page
- [ ] Favicon live
- [ ] Carousel template (3 frames)
