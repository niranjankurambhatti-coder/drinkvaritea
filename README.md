# Drink Varitea

Master operating repository for the Drink Varitea tea ecommerce company.

Drink Varitea is a taste-first DTC tea brand focused on premium but accessible tea discovery, customizable tea experiences, rewards, and fast growth experimentation.

## Operating thesis

Build a tea company as an experiment engine:

- Better UI/UX than incumbent tea brands
- Faster SEO and content loops
- Paid acquisition with strict CAC/ROAS guardrails
- Interactive quiz and custom tea-builder experience
- Rewards and dynamic deals
- Data-backed product, bundle, and pricing decisions

## Repository purpose

This repo is the source of truth for:

- Agent boundaries and operating contracts
- Web app and landing page architecture
- Analytics and event tracking plan
- SEO and paid acquisition loops
- Social automation stack
- Experiment backlog and result logs
- Connector setup checklist
- Product catalog transformations
