Layer 1 of the pipeline. Re-rendering any stage requires a written ADR and a stage version bump.

The locked landing hero

Names live in an HTML overlay, never baked into the image. Localizable, A/B-testable, editable in seconds.

Brand kill rules

Lines we do not cross.

  • No mascots, ever.
  • Sentence case only on pouches, labels, and UI.
  • Matte everywhere except tea leaves, brewed tea, glass jars, wet botanicals.
  • No health, origin, sourcing, or certification claims.
  • No AI-generated text on any branded surface.
  • Every scene ships desktop and mobile variants from the same template family.
  • Re-rendering a locked stage requires a written ADR.
