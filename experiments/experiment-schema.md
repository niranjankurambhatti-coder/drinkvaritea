# Experiment Schema

Use this schema for every experiment.

| Field | Definition |
|---|---|
| Experiment name | Short searchable name |
| Owner agent | Agent responsible |
| Date started | Start date |
| Date ended | End date |
| Hypothesis | If we do X for Y audience, Z metric will improve because... |
| Audience | Persona or segment |
| Channel | SEO, paid search, paid social, email, web, social organic |
| Offer | Product, bundle, discount, quiz, builder, or content |
| Variant A | Control |
| Variant B | Test |
| KPI | Primary metric |
| Guardrail | Metric that must not degrade |
| Threshold | Success/failure threshold |
| Result | Numbers |
| Decision | Scale, iterate, pause, kill |
| Learning | What changed in company knowledge |
